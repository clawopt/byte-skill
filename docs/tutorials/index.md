# 技术教程 · 模块总览

这里是 ByteTech 的「技术教程」总入口，你可以从不同模块中选择适合当前阶段的内容：

## C++ 教程

- 路径：`/tutorials/cpp/`
- 主要内容：
  - C++ 语言定位与典型应用场景
  - 开发环境与编译工具链
  - 在线代码调试体验（网页内嵌 C++ 运行环境）

> 适合想要了解底层性能与工程实践的同学。

## 操作系统教程

- 路径：`/tutorials/os/`
- 主要内容：
  - 操作系统在软硬件之间的角色
  - 进程/线程、内存管理、文件系统等核心概念
  - 从 Linux 实际行为理解抽象概念

> 适合希望从工程视角理解「程序如何真正跑起来」的同学。

## 计算机网络教程

- 路径：`/tutorials/network/`
- 主要内容：
  - 一次 HTTP 请求背后完整的网络传输路径
  - TCP/IP 协议栈与分层模型
  - DNS、CDN、反向代理与常见网络排障思路

> 适合需要提升系统设计与线上排障能力的同学。
