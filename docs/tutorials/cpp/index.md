# 第 1 章 · C++ 语言与环境简介

C++ 是一门兼具 **高性能** 与 **高抽象能力** 的编程语言，在游戏引擎、高性能服务器、数据库内核、嵌入式等领域都有广泛应用。本教程面向有一定编程基础、希望向「工程级 C++ 开发」迈进的同学。

在本章中，你将了解：

- C++ 在现代软件栈中的定位
- C++ 标准与编译器生态（GCC/Clang/MSVC 等）
- 基础开发环境的搭建方式
- 如何在网页中快速体验、调试一段 C++ 代码

## 1.1 C++ 的典型应用场景

- **系统软件**：操作系统内核、驱动、数据库内核（如 MySQL/InnoDB 的大量核心代码使用 C++ 实现）。
- **高性能服务端**：高并发网关、撮合引擎、实时计算等对延迟极度敏感的服务。
- **桌面与游戏开发**：游戏引擎（Unreal Engine）、图形渲染、桌面客户端等需要精细控制资源的场景。
- **嵌入式与 IoT**：资源受限但又需要一定抽象能力的设备端开发。

理解这些场景，有助于你在后续学习中建立「为何而学」的直觉。

## 1.2 本地开发环境概览

日常工程开发中，你通常会在本地安装：

- 一个 C++ 编译器：如 `g++`（GCC）、`clang++` 或 MSVC。
- 一个熟悉的编辑器或 IDE：如 VS Code、CLion、Visual Studio 等。
- 构建工具：如 CMake、Ninja 或各类工程脚手架。

本教程后续章节会围绕「如何编写现代 C++ 代码」展开，而不是停留在语法罗列上。
