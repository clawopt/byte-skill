## 后端开发工程师 · 数据库开发工程师学习路线概览

> 下面是数据库开发工程师的典型成长路径，你可以按顺序学习，也可以按模块查漏补缺。

1. **数据库基础与概念**（了解数据库在后端系统中的角色与整体架构）  
2. **数据库原理**（事务、并发控制、日志、多版本等核心概念）  
3. **存储引擎**（页、索引、缓存、刷盘策略等）  
4. **SQL 与索引优化**（执行计划、慢查询分析、索引设计）  
5. **分布式与高可用数据库**（主从复制、分片、中间件、CAP、共识协议）  
6. **性能调优与排障**（监控指标、热点问题、锁等待、死锁排查）  
7. **项目实践与面试准备**（从真实业务需求出发设计数据库方案）  

在左侧侧边栏中，你可以进入每一阶段的专业教程页面，查看更细致的学习指引与章节内容。
